Welcome!

This is a working demo of an online phonetic adaptation experiment, written entirely in HTML, CSS, and JavaScript (with liberal use of JQuery). If you want to play around with the code yourself, fork it over on Bitbucket.

The actual experiment is a replication and extension of Vroomen, van Linden, de Gelder, and Bertelson (2007) [pdf]. We included two extra conditions, which you can read all about in Kleinschmidt and Jaeger (2012) [link].

If you just want to run the demo (really the full experiment), choose which condition to run:

  • S hands
  • SH hand
  • S mouth
  • Sh mouth

You can also skip the calibration phase and go straight to the adaptation part.
